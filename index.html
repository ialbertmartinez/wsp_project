Web Components Challenge

Design + Code by Albert Martinez

A Full-Stack Developer that loves creating functional web solutions and engaging digital experiences. This was a personal project to push the limits of my abilities as a designer and developer.

The Challenge

The challenge was to construct 10 web components from scratch, but The design could not be my own.
The Design came from any website template online. Took a screenshot of a component i liked and moved on to the next 9 to repeat the above steps.

10 different components from 10 different websites. I used:

  • TemplateMonster
    •
  • UI8
    •
  • AWWWAwards
    •

Requirements

Each component must:

  • Be coded from scratch without the use of frameworks (Font-Awesome and Google Fonts is ok!)

  • Be written in HTML & CSS only
  • Meet current W3C Standards by verifying the code with the W3C validator tool
  • Be responsive
  • match as much as possible like the screenshot taken of the corresponding component
