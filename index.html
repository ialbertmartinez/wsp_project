About me

Hello, there. I'm Albert Martinez.
A Full-Stack Developer that loves creating functional web solutions and engaging digital experiences.

Purpose

I created a library of 10 fully functional components built from scratch. I wanted to test my skills and abilities as a front-end developer and a UI designer.

Goal

Construct 10 web components from scratch. Share my learninig experience with the WWW to help any who may be looking for some info on coding web components from scratch, web standards, or responsive web design info.

Requirements

Each component will be made from its screenshot. Screenshot will be the only reference from which to build upon. I created some extra rules to spice things up a bit:

    Code must:

  • Be original
  • Written without use of any 3rd party libraries
  • Be in HTML & CSS only
  • Meet current W3C Standards. The W3C validator tool was used for validation
  • Be responsive
  • Look like the screenshot taken of the corresponding component
